Projeto para cálculo de Prazo e Frete, dado parâmetros como custo, rota e peso.


=============== AMBIENTE =======================
Para execução deste projeto utilizamos as seguintes estruturas

VirtualEnv
https://virtualenv.pypa.io/en/stable/ (para trabalhar num ambiente isolado com as respectivas bibliotecas)

Instalação de pacotes (uma vez instalado e criado o VirtualEnv específico) :

pip install -r requirements.txt

Execução do sistema : 

meli.py saopaulo florianopolis 50 130
